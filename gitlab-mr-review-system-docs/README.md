# GitLab MR Review System Docs Bundle

本目录包含可直接进入开发阶段的文档集：

- `research.md`：扩范围调研总报告
- `prd.md`：产品需求文档
- `security.md`：安全策略与边界
- `api-contracts.md`：关键接口与 schema
- `evaluation.md`：评估方案
- `openspec/0001-gitlab-mr-review-system/spec.md`：系统规格
- `openspec/0001-gitlab-mr-review-system/design.md`：技术设计
- `openspec/0001-gitlab-mr-review-system/tasks.md`：任务拆解

推荐阅读顺序：

1. `research.md`
2. `prd.md`
3. `openspec/.../spec.md`
4. `openspec/.../design.md`
5. `security.md`
6. `api-contracts.md`
7. `evaluation.md`
8. `openspec/.../tasks.md`
